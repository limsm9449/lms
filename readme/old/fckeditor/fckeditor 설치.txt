* FCKeditor 설정

1.FCKeditor_2.6.10\fckeditor 를 webapp에 추가해준다.

2.fckeditor-java-2.5-bin.zip에 있는 jar 을 WEB-INF\lib에 복사해준다.
	commons-fileupload-1.2.1
	commons-io-1.3.2
	fckeditor-java-core-2.5
	imageinfo-1.9
	
3. web.xml에 다음을 추가한다.
   <servlet>
        <servlet-name>ConnectorServlet</servlet-name>
        <servlet-class>
            net.fckeditor.connector.ConnectorServlet
        </servlet-class>
        <load-on-startup>1</load-on-startup>
    </servlet>

    <servlet-mapping>
        <servlet-name>ConnectorServlet</servlet-name>
        <url-pattern>/fckeditor/editor/filemanager/connectors/*</url-pattern>
    </servlet-mapping>

4.servlet-context.xml에 다음을 추가한다.
   url이 fckeditor 인 html은 fckeditor로 가도록 해준다.
   
	<resources mapping="/fckeditor/**" location="/fckeditor/" />

5.fckeditor.properties 파일을 WEB-INF\classes에 넣어준다.
