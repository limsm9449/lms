동영상 설정 동영상 설정 안내입니다.

사용중인 WMP 버전에 맞는 환경설정을 통해 동영상 재생 문제를 해결할 수 있습니다.
큐러닝은 Windows Media Player 9 버전 이상에서 최적화 되어 있습니다.
본인의 Windows Media Player 버전 확인 방법은 상단의 메뉴 중 [도움말] > Windows Media Player 정보에서 확인 가능합니다.
※ 상단의 메뉴 모음이 보이지 않는 경우, 키보드 좌측 하단의 Alt 키를 눌러 주세요.

공통 조치사항

Step 1. Windows > 시작 > 실행 클릭 (단축키 : 윈도우키 + R)

Step 2. "regsvr32 wmnetmgr.dll" 을 입력하고 [확인] 을 클릭합니다.

Step 3. "...성공했습니다." 메세지가 나오면 [확인] 을 클릭합니다.

Windows Media Player12 상세 설정 안내

Step 1.

①번 Windows Media Player의 [도구] 클릭
②번 [옵션] 클릭
※ 상단의 메뉴 모음이 보이지 않는 경우, 키보드 좌측 하단의 Alt 키를 눌러 주세요.

Step 2.

①번 [플레이어] 탭 클릭
②번 '인터넷에 연결(다른 명령 무시)' 체크

Step 3.

①번 [네트워크] 탭 클릭
②번 'RTSP/UDP' 체크 해제, 'RTSP/TCP', 'RTSP/HTTP' 에 체크
③번 '플레이어에서 멀티캐스트 스트림 수신 허용' 에 체크

④번 [구성] 버튼을 클릭

⑤번 '프록시 서버 사용 안 함' 에 체크

Step 4.

①번 [성능] 탭 클릭
②번 'WMP 파일용 DirectX 비디오 가속 켜기' 의 체크 해제

Step 5.

①번 Windows > 시작 > 기본프로그램 클릭
②번 Windows Media Player 클릭
③번 '이 프로그램을 기본 프로그램으로 설정' 클릭 후 [확인] 버튼 클릭

④번 '이 프로그램에 대한 기본값 선택' 클릭
⑤번 '모두선택' 체크 후 [저장] 버튼 클릭

공통 조치사항

Step 1. Windows > 시작 > 실행 클릭 (단축키 : 윈도우키 + R)

Step 2. "regsvr32 wmnetmgr.dll" 을 입력하고 [확인] 을 클릭합니다.

Step 3. "...성공했습니다." 메세지가 나오면 [확인] 을 클릭합니다.
